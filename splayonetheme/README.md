Splay One WP Theme
===================

Bootstrap (http://getbootstrap.com) in WordPress theme form.

GETTING STARTED
_______________

To get started, open Terminal or a command prompt and run:

	cd path/to/wp-content/themes
	git clone https://github.com/hadlabf/cloudways-splay-theme.git
	npm install
	grunt dev
